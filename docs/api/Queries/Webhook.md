# Webhook query

Get one single webhook by id.

### Code method
`\Automattic\WooCommerce\Api\Webhooks::get_webhook`

### REST route
`GET /wp-json/wc/v4/rest/webhook`

## Return type
[`Webhook`](../ObjectTypes/Webhook.md)

## Arguments

| Name | Type | Description |
|-|-|-|
id | `int!` | Id of the webhook to get.
